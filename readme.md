# Fitbase Parser

Скрипт для автоматического сбора данных об абонементах, услугах и заморозок клиентов из системы Fitbase (encoreiset.fitbase.io).

## Описание

Скрипт автоматизирует процесс сбора информации об абонементах, услугах и заморозок клиентов:
- Авторизуется в системе через Selenium
- Получает cookies для последующих API-запросов
- Собирает данные о клиентах и их абонементах
- Сохраняет результаты в Excel-файл

## Возможности

- Автоматическая авторизация с ручным вводом капчи
- Сбор данных через API-запросы (быстрее, чем через Selenium)
- Фильтрация данных по дате
- Формирование отчетов

## Требования
- Google Chrome браузер
- ChromeDriver (совместимый с версией Chrome)
